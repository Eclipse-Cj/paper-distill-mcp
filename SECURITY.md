# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Instead, please report it privately via:

- GitHub: [Security Advisories](https://github.com/Eclipse-Cj/paper-distill-mcp/security/advisories/new)
- Email: vertex.cj@gmail.com

We will respond within 48 hours and work on a fix before public disclosure.
